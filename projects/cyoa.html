CYOA Project

Context

For my CYOA project, I collaborated with my partner, Johnny, to create a file path in which users can choose their own adventures by selecting their desired choices at certain levels. This project mainly involved using markdown on GitHub.

Content

To start with the project, my partner and I first designed a plan. Given our topic, "The Game Of Life", my partner and I figured it was best to keep most of the file paths and selection choices similar to the reality of the world.
